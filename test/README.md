# Project: ibe-client-ui (e.g. Web 4.0)

## Quick Start

### Prerequisite

- [Node](https://nodejs.org/en/) v0.12.4 or higher

### Current Build Process:
* npm install
* gulp
* npm start
* open localhost:3000

### Development Build Process:
* npm install
* gulp watch
* nodemon server
* open localhost:3000
